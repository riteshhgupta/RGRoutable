# RGRoutable

## Installation

### Cocoapods

To integrate RGRoutable into your Xcode project using CocoaPods, specify it in your Podfile:

```
pod 'RGRoutable', :git => 'https://github.com/riteshhgupta/RGRoutable.git'
```
